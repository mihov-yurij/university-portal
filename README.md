# ⚓ University Portal - ONMU

Современный веб-портал для Одесского национального морского университета (ОНМУ). Проект ориентирован на высокую производительность, адаптивность и удобство пользователей.

## 🚀 Стек технологий

*   **Framework:** [React 18](https://reactjs.org)
*   **Build Tool:** [Vite](https://vitejs.dev) — для мгновенной сборки и HMR.
*   **Language:** [TypeScript](https://typescriptlang.org) — строгая типизация для надежности кода.
*   **Styling:** [Tailwind CSS](https://tailwindcss.com) — утилитарный подход к верстке.
*   **Icons:** [Lucide React](https://lucide.dev) — легкие и гибкие иконки.
*   **Deployment:** Оптимизировано для [Vercel](https://vercel.com).

## ✨ Особенности

- 📱 **Mobile First:** Полная адаптивность под любые устройства.
- ⚡ **Performance:** Высокие показатели Core Web Vitals благодаря Vite.
- 🧩 **Modular:** Компонентная архитектура, готовая к масштабированию.
- 🛠 **Clean Code:** Настроенный ESLint для поддержания единого стиля разработки.

## 📦 Быстрый старт

### Требования
*   Node.js (версия 18.x или выше)
*   npm или yarn

### Установка и запуск
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com
   ```

2. Перейдите в папку проекта:
   ```bash
   cd university-portal
   ```

3. Установите зависимости:
   ```bash
   npm install
   ```

4. Запустите сервер для разработки:
   ```bash
   npm run dev
   ```

## 🏗 Структура проекта

- `/src` — исходный код приложения.
- `/public` — статические ресурсы (логотипы, шрифты).
- `vercel.json` — конфигурация для корректной работы SPA-роутинга при деплое.
- `tailwind.config.js` — кастомные настройки темы и стилей.

## 🛠 Скрипты

- `npm run dev` — запуск локального сервера.
- `npm run build` — сборка проекта для продакшена.
- `npm run lint` — проверка кода линтером.
- `npm run preview` — предпросмотр собранного билда.
