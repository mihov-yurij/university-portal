# ⚓ University Portal - Кафедра «Морський Бізнес та Маркетинг» ОНМУ

Современный веб-портал для кафедры Одесского национального морского университета. Проект ориентирован на абитуриентов, студентов и преподавателей, предоставляя удобный доступ к учебным материалам и официальной информации.

## 🚀 Технологический стек
*   **Frontend:** React 18 + TypeScript
*   **Сборка:** Vite
*   **Стилизация:** Tailwind CSS (адаптивная верстка)
*   **Иконки:** Lucide React
*   **Деплой:** Vercel

## ✨ Особенности
*   **Адаптивность:** Полная поддержка мобильных устройств с удобным "бургер-меню".
*   **Динамическая навигация:** Многоуровневое выпадающее меню с удобной группировкой разделов.
*   **SEO-friendly:** Чистая структура заголовков и мета-данных.
*   **SPA-routing:** Настроенные редиректы на Vercel для бесперебойной работы навигации.

## 📦 Запуск проекта
1. Клонировать репозиторий
2. Установить зависимости: `npm install`
3. Запустить локально: `npm run dev`
4. Собрать билд: `npm run build`
